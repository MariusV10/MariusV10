Informatik Noter

HTML står for Hyper Text Markup Language. HTML er et form for standard sprog man bruger til at etablere hjemmesider. HTML beskriver strukturen af en hjemmeside. HTML består af en række elementer der har til formål at fortælle hjemmesiden hvordan den skal vise det input man laver (f.eks. baggrunden på hjemmesiden).

CSS står for Cascading Style Sheets. CSS beskriver hvordan HTML elementer bliver vist på skærm, papir eller andre medier. CSS er en god og meget praktisk løsning. Den kan kontrollere layotet på flere forskellige hjemmesider på én gang. Hvordan indsætter man en CSS? Det er der tre måder på. External CSS, Internal CSS og Inline CSS
